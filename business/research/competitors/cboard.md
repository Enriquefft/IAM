# What it is
Free, open-source web-based AAC, started by an Argentinian developer (Shay Cojocaru) and incubated under UNICEF Office of Innovation. Browser-first, also packaged as Android, iOS, Windows. Designed explicitly to fix the 4-figure-cost problem of mainstream AAC apps.

# Pricing
- Free, fully featured, no ads, no upsells.
- AGPL-3.0 codebase; donations via OpenCollective.

# Features
- Pictogram-based grid AAC; uses Web Speech API for TTS in browser.
- Symbol library from [ARASAAC](arasaac.md), OpenSymbols, Mulberry.
- Editable home boards, custom uploads, search through pictogram library.
- Adaptive personalization: tailoring of content suggested per learner.
- Offline support on Chrome (desktop + Android).
- Cloud sync with account.
- Cross-platform: web, Android (Play Store), iOS (App Store), Windows.
- 46 languages in UI (machine-translated; some need proofreading).

# Strengths
- Truly free at the point of use — wins on cost in markets where USD 250 AAC apps are non-starters.
- ARASAAC integration is native — Spanish symbol-set is first-class.
- UNICEF endorsement opens public-sector and NGO channels.
- Web-first means it runs on a sub-USD 100 Android tablet.
- Open source with active GitHub repo.

# Weaknesses / gaps
- No data collection, no progress tracking, no parent dashboard.
- TTS quality depends on browser/OS — can be poor on cheap Android.
- No vocabulary scaffolding (no Crescendo / Unity equivalent) — the user is handed a blank board and a symbol search.
- Machine-translated UI in many languages — Spanish is OK but not native-quality copy.
- No marketing presence: families discover Cboard via SLPs or news articles, never via ads or app store ranking.
- No support, no curriculum, no training program. SLPs needing a class to teach the app find none.
- No video modeling, social stories, or schedule features.

# Evidence base
- No published clinical trials specific to Cboard.

# Geographic / language coverage
- 46-language UI; native-quality Spanish, English, Portuguese.
- Distributed worldwide via web + app stores.
- LATAM-origin (Argentina), so default branding and onboarding is in Spanish.

# Usage signal
- ~100K cumulative installs on Google Play since April 2020 (Aptoide listing; Similarweb tracks it). ~6 years of distribution → roughly 1.4K installs/month average — small for a free, UNICEF-endorsed AAC.
- Almost no visible Google Play user reviews → engagement past install appears low. Typical Play review rate in this category is 0.5–2% of installs; <0.1% is a tell of weak retention.
- Active development as of 2026-02-27 (v1.39.0): symbol-search tour, ElevenLabs voice integration, crash fixes.
- Communication-category Play ranking is mid-pack outside Spanish-speaking markets (Israel #133, Australia #268, Poland #277, Canada #364, Brazil #418 — Similarweb, 2026-04-30). Notable absence from Spanish-LATAM top rankings despite Argentinian origin.
