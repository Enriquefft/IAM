# What it is
Danish-origin ABA clinical platform built around the kid (rather than the practice). Data collection, customizable programs, telehealth, AI documentation, supervision tools, parent training. Designed for cross-border ABA delivery (Europe + US).

# Pricing
- Per-client billing.
- Standard — USD 19/client/month, includes Balance + 1 hr/month telehealth/client.
- Premium — USD 24/client/month, includes 30 hr/month telehealth, live supervision, audit logs, pre-made program library.

# Features
- DTT, target sets, task analysis, interval, duration, rate, ABC.
- Mix-and-match data types in one program.
- Real-time entry.
- AI documentation.
- Telehealth (BCBA-RBT and BCBA-parent).
- API for integration with PM/billing/HR.
- Multi-language UI: English + several European languages.

# Strengths
- Per-client pricing model rewards smaller clinics.
- Telehealth-built-in is a real differentiator vs CentralReach (where telehealth is added).
- API-first integrations.
- Supervision tools designed for cross-border BCBA model.
- Multilingual.

# Weaknesses / gaps
- Smaller install base than CentralReach.
- Per-client billing surprises clinics with growing caseloads.
- ABA-only.
- Spanish UI present but training/marketing in English.
- Parent training tooling is therapist-mediated; no direct parent product.
- No LATAM presence.

# Evidence base
- Tooling, not clinical claim.

# Geographic / language coverage
- Multiple EU languages + English.
- Sold in US, EU, UK.
