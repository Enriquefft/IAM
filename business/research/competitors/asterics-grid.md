# What it is
Free, open-source web AAC out of UAS Technikum Wien (Austria), under the AsTeRICS Foundation. Browser-based grid communicator with input flexibility (touch, mouse, switch, eye gaze, head tracking) — built around the lab's accessibility hardware ecosystem.

# Pricing
- Free, no ads, AGPL-3.0.
- Donations via OpenCollective.

# Features
- Browser-based dynamic grid AAC; PWA, can be installed offline.
- [ARASAAC](arasaac.md) API integration for symbol search (live since 2022).
- Open Symbols, Mulberry libraries.
- Multiple input modalities: touch, mouse, keyboard, scanning, gaze, head tracking.
- Smart-home control bindings (turn lights, control media) — uniquely targets adult communicators with motor disabilities.
- Translation of grids into any language (auto-translate of own boards).
- Cloud sync (own server or self-hosted).
- 10+ UI languages including Spanish.

# Strengths
- Accessibility breadth — handles severe motor disability cases that touch-only apps don't.
- Active ARASAAC partnership; Spanish pictogram coverage is best-in-class for free tools.
- Open source, self-hostable, GDPR-friendly.
- Smart-home integration is a wedge into adult AAC market that no consumer app touches.
- Backed by an academic foundation; updates are continuous.

# Weaknesses / gaps
- Web-first UX feels less smooth than native apps; loading states visible.
- Setup requires understanding "grid" concepts; very steep for parents.
- No vocabulary curriculum bundled — user starts close to blank.
- No data tracking, no parent dashboard, no skills progression.
- Documentation is technical — many videos in German.
- Brand awareness very low outside DACH / academic AAC.
- TTS quality depends on browser.

# Evidence base
- Academic publications on AsTeRICS framework (assistive accessibility); no clinical RCTs of the Grid product.

# Geographic / language coverage
- 10+ UI languages including Spanish, English, German.
- ARASAAC integration provides Spanish-first symbol library.
- Free worldwide via web.
