# What it is
iOS visual schedule app from Good Karma Applications. Builds first-then boards, longer schedules, task analyses, social stories, choice boards, video models — broad scope for one app. Targeted at kids with autism and developmental delays.

# Pricing
- First Then Visual Schedule (basic) — USD 9.99 one-time.
- First Then Visual Schedule HD (full) — USD 14.99 one-time.
- "First Then Board" (online platform from same publisher) — separate web pricing.

# Features
- First-then board (2 cells) plus longer sequences.
- Image sources: device camera, camera roll, web search, built-in stock library, 10,000 SmartySymbols pictograms.
- Video modeling per board.
- Choice boards.
- Social stories builder.
- Multi-user.
- iOS universal binary.

# Strengths
- Strongest single-app coverage of OT-style visual supports — first-then through full schedule through video model in one place.
- One-time price.
- 10,000 SmartySymbols built in.

# Weaknesses / gaps
- iOS only.
- English UI and stock images only.
- No data, no analytics, no clinician dashboard.
- Limited cloud / sync between devices.
- Basic / HD split is confusing for buyers.
- Slow release cadence; UX shows it.
- "First Then Board" web product is a fork that adds online team collaboration but is sold as a separate subscription — fragmented offering.

# Evidence base
- Anecdotal; no RCTs.

# Geographic / language coverage
- English.
- Sold via App Store.
