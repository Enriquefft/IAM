# What it is
Tobii Dynavox's flagship AAC software. Symbol-based grid AAC built on the PCS (Picture Communication Symbols) library. Available as a subscription on iOS App Store, Android (newer), Windows, and pre-loaded on dedicated Tobii Dynavox SGD hardware. Backed by the largest dedicated AAC hardware company (Sweden-origin, public).

# Pricing
- TD Snap iOS subscription — model varies; contact-for-pricing on Tobii portal. `Estimate:` USD 199-300 first year per public reseller pages.
- Pre-loaded on Tobii Dynavox dedicated hardware — devices priced USD 4,000-15,000, typically funded through Medicare / private insurance / VA in US, NHS in UK.
- Boardmaker 7 subscription (companion software) — from USD 99/year per learner.
- Compass (legacy software) — perpetual + subscription options.

# Features
- PCS pictogram library (industry standard).
- Page sets: Core First, Motor Plan, Aphasia, Visual Scene Display.
- Eye-tracking integration with Tobii Dynavox eye-gaze hardware.
- Switch scanning, head tracking.
- Bilingual EN/ES.
- Communication partner training material via Boardmaker.

# Strengths
- Vertical integration (hardware + software + funding facilitation) with the deepest reach into US insurance reimbursement.
- The eye-gaze story — for users with severe motor + speech disability, only Tobii Dynavox seriously competes.
- Boardmaker is the legacy gold standard for AAC printed materials in US schools.
- Multiple page sets cover wide motor and cognitive range.

# Weaknesses / gaps
- Pricing opacity is severe; hardware locks customers into a 5-figure capital outlay before software seats.
- Insurance funding model is US/UK-centric. In LATAM private-pay only.
- PCS library is paid and proprietary — competing apps cannot use it.
- UI is enterprise-grade, not parent-friendly.
- No data product around outcomes; software is output-only.
- Subscription migration has frustrated long-time Boardmaker users.

# Evidence base
- PCS library has decades of clinical use; numerous studies of AAC method, fewer of the Tobii Dynavox software itself.

# Geographic / language coverage
- EN, ES, FR, DE, IT, PT, NL, more.
- Distributed in US, UK, EU, AU, parts of LATAM via resellers.
