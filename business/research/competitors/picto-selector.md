# What it is
Free Windows desktop application by Martijn Mulder (Netherlands). Built explicitly for printing visual supports — schedules, choice boards, semantic maps, behavior rules, bingo cards. Pulls from 30,000+ ARASAAC pictograms plus uploadable images. Used by tens of thousands of teachers, SLPs, and parents internationally.

# Pricing
- Free. Windows only. Donation-supported.

# Features
- 30,000+ ARASAAC pictograms (full database).
- Pictogram Maker: combine multiple pictograms into a custom one.
- Templates for schedules, semantic maps, calendars, bingo, behavior rules.
- Multilingual labels: Spanish, English, Dutch, German, French, Danish, Portuguese, more.
- Export to print, PDF, or image.
- Library management for own and ARASAAC pictograms.

# Strengths
- Best-in-class for the print workflow — the parent prints a paper schedule, the kid uses it on the fridge or in school.
- Long-running, actively maintained.
- Volunteer translation community keeps multi-language labels current.
- Fully free with no business pressure.
- Fits into school and clinic workflows where laminated paper supports are still preferred.

# Weaknesses / gaps
- Windows only. No Mac, no mobile, no web.
- Authoring tool, not an end-user app — assumes the supporter is the active user.
- No interactivity; pictogram has no TTS in the printed artifact.
- No user accounts, sync, or analytics.
- Heavy UI; Windows-90s style.
- Single developer; no commercial roadmap.

# Evidence base
- None; tooling.

# Geographic / language coverage
- Multi-language pictogram labels; Spanish is well-covered.
- Used worldwide.
