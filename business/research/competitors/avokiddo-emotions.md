# What it is
Open-ended creative play app from Avokiddo. Three animal characters; the kid attaches 100+ props and watches the characters react with different emotions. Designed to teach cause-and-effect for emotion. Recommended for autistic and non-autistic preschoolers.

# Pricing
- Premium one-time IAP — `Estimate:` ~USD 2.99-4.99 (varies by store).
- No ads, no third-party content.

# Features
- 3 animal characters.
- 100+ interactive props.
- Open-ended exploration; no goals or scores.
- Characters react with named emotions.
- iOS and Android.

# Strengths
- Open-ended, no-fail design fits autism-acceptance pedagogy: kid explores at own pace.
- Cause-and-effect with emotional outcome is the right pedagogical handle.
- No ads, no IAPs in main flow, no PII collection.
- Cross-platform.
- Friendly visual design.

# Weaknesses / gaps
- Single-mechanic app; replay value tapers.
- English only (text labels for emotions).
- Discoverability issues — Avokiddo is a small studio.
- No data, no progress tracking — kid plays, parent has no signal of learning.
- Not a curriculum.
- Casual play, not therapy.

# Evidence base
- None published.

# Geographic / language coverage
- English UI.
- iOS + Android worldwide.
