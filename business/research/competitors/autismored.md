# What it is
LATAM-origin (Argentina) free social-network-style platform for the autism community. Geolocation search to connect families with professionals, autism-friendly services, NGOs, businesses (medical, psychology, therapy, legal, hairstylist, workshops). Built by Rotary Club Montserrat with Fundación Brincar, APADEA, PANAACEA, and Globant.

# Pricing
- Free.

# Features
- Geolocation-based directory of autism-friendly services and professionals.
- User profiles: "natural person" (family/patient) or "service/resource".
- Community: news, events, conversations.
- Currently 7,600+ users (per recent press).

# Strengths
- LATAM-first, Spanish-first.
- Backed by major Argentinian autism orgs (APADEA, PANAACEA, Brincar) plus Rotary network and Globant tech contribution.
- Geolocation directory solves a real LATAM pain (services are scarce and dispersed).
- Free.

# Weaknesses / gaps
- Directory liquidity is the entire game; 7,600 users is small for the LATAM market.
- Argentina-heavy listings; coverage thinner in MX/CO/PE/CL/EC.
- No tools for the kid — purely a parent / professional service-discovery layer.
- No structured intake or matching algorithm beyond geolocation.
- No reviews or quality signals on listed providers.
- Funding model unclear long-term.

# Evidence base
- Network product; not a clinical claim.

# Geographic / language coverage
- Spanish (LATAM).
- Web platform; mobile-friendly.
