# What it is
Augmentative communication system from Fundación Orange (designed by Tropical Studio with SLPs José Manuel Marcos and David Romero). Distinct from pictogram-only systems: uses photographs as a bridge for users who don't yet abstract pictograms. Includes a free web platform with images and printable boards plus a companion app with structured exercises (sentence construction, vocabulary by semantic field, classification, association).

# Pricing
- Free. Web platform + iOS + Android. No ads, no IAPs.

# Features
- Photographic image library (real photos rather than line-drawn pictograms).
- Printable boards.
- Companion app with structured exercises: sentence building, vocabulary, classification, association.
- Created with input from professional SLPs.

# Strengths
- The photograph-based approach fills a genuine clinical gap — autistic learners who don't recognize pictograms but recognize photos. Most AAC systems skip this scaffolding.
- Free.
- Spanish-first.
- Backed by Fundación Orange (long-term funding).
- SLP co-design gives clinical credibility.

# Weaknesses / gaps
- Catalog of photos is fixed; cannot grow with user-uploaded photos in the app to the same standard.
- Web + app feels split; user has to discover both.
- No data tracking, no user accounts.
- Spanish only.
- App functionality is exercise-focused; not a full AAC system the user can carry around.
- Targets early communicators; little for advanced.

# Evidence base
- SLP co-designed; no published RCT.

# Geographic / language coverage
- Spanish.
- Free worldwide.
