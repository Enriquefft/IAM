# What it is
Symbol-based AAC app for non-verbal users, the long-incumbent "Cadillac" of AAC. User taps cells with pictograms (Crescendo or SymbolStix library), the app speaks the word/sentence with a TTS voice. Heavy customization: cell size, vocabulary level, motor planning grid, switch-scan access. Apple-only ecosystem.

# Pricing
- Proloquo2Go (legacy one-time) — USD 249.99 one-time, App Store. Still listed but being migrated to subscription.
- Proloquo (new SaaS, separate app) — USD 9.99/month or USD 99.99/year per user, with 50% off year one for new users.
- Crescendo Evaluation — free 30-day trial via separate "Proloquo2Go Discover" app.

# Features
- 25,000+ symbol library (Crescendo, SymbolStix add-on).
- 100+ TTS voices including child voices (Acapela).
- Bilingual mode: switch between English / Spanish / French / Dutch on the fly inside one user profile.
- Switch access, eye-gaze (via accessibility shortcuts), keyguard support.
- Word prediction, grammar support (verb conjugation, plurals).
- Pre-built grid sets at three vocabulary levels (Basic Communication, Intermediate, Advanced Core).
- Backup/restore via iCloud and Dropbox.

# Strengths
- Largest install base in AAC; SLPs default to recommending it; therefore third-party training material is abundant.
- Apple Watch companion for emergency phrases.
- Crescendo vocabulary is genuinely research-informed (core word approach).
- True bilingual ES/EN, not just UI translation — symbols and grammar localize.

# Weaknesses / gaps
- iOS / iPadOS / macOS only. No Android, no web. Locks families into ~USD 400+ hardware before software cost.
- Migration to subscription model (Proloquo, separate app) has confused users and split the product.
- No data collection for the parent or therapist — speech output is generated but never logged to a session record. Other tabs of the user's life (mood, schedule) live in other apps.
- Setup is overwhelming for parents without an SLP; defaults are advanced-communicator oriented.
- LATAM presence is incidental — language is supported but no local distribution, training, or insurance reimbursement path. Spanish voices default to Castilian.
- No emotion/regulation features. No visual schedule. No video modeling. Pure speech output.

# Evidence base
- Crescendo vocabulary is built on core word research (Banajee, Dennis & Stuart, etc.) — see `clinical/topics/aac-core-vocabulary.md` when populated.
- AssistiveWare publishes case studies but not RCTs of the app itself.

# Geographic / language coverage
- True L10n: English, Spanish, French, Dutch (UI + symbol labels + voices + vocabulary grammar).
- Sold worldwide via Apple App Store. Funding routes (insurance, education) only mature in US, UK, NL, AU.
