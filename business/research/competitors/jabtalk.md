# What it is
Free Android-only AAC app, made by a parent of an autistic child as a deliberate counter to the absence of free AAC on Android in 2012. Minimal, photo-and-recording-based; not symbol-library-based. The user records audio for each cell, attaches a photo or imports an image.

# Pricing
- Free. No ads, no IAPs.

# Features
- Picture + recorded audio cells (or text-to-speech).
- Use device camera to capture pictures inline.
- Import photos and audio files from device storage.
- Multi-level grid navigation.
- Backup/restore as ZIP.

# Strengths
- Genuinely free with no business model behind it; download-and-go for any Android phone or tablet.
- Photo-first design is more accessible than symbol libraries for many beginners — especially in cultures where [ARASAAC](arasaac.md) pictograms feel foreign.
- ~150K cumulative Google Play installs (AppBrain, 2026-05-02); 3.90/5 from ~280 ratings — a small but durable parent following.
- Small enough to run on the cheapest Android tablet.

# Weaknesses / gaps
- Android only.
- No symbol library — everything must be photographed or imported. Slow to set up.
- No vocabulary scaffolding, no core-word system, no grammar.
- UI is dated and developer-grade; no localization beyond what users contribute.
- No cloud sync; local backup only.
- No data tracking, no parent or therapist dashboard.
- No active development cadence; effectively a finished hobby project.
- No clinical evidence base.

# Evidence base
- None published.

# Geographic / language coverage
- UI primarily English; community translations exist.
- Free worldwide via Google Play.

# Usage signal
- ~150K cumulative installs since ~2012 launch → average ~1K installs/month over its lifetime. Long tail, no growth.
- Rating 3.90/5 from ~280 ratings (AppBrain, 2026-05-02). Active raters are a tiny fraction of installs.
- No release cadence visible since the founding parent finished the app — effectively in maintenance mode.
