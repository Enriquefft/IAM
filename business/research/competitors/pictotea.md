# What it is
Free Argentinian Android AAC app from Fundación Qualis. Pictogram-based grid with five learning stages of increasing complexity. TTS reads each pictogram and the constructed sentence aloud. Designed to be a no-cost first AAC step.

# Pricing
- Free. Android only.

# Features
- 5 progressive levels of difficulty.
- Symbol-tap to TTS playback.
- Sentence builder (chain pictograms, then play).
- User-customizable: add own pictograms.

# Strengths
- Free, made in LATAM, distributed for LATAM Spanish-speaking families.
- Five-stage curriculum is a real differentiator over Cboard's blank-slate approach.
- Short, simple UX — low cognitive load for parents.

# Weaknesses / gaps
- Android only; iOS users are locked out.
- Visibly low-budget UX; graphics dated.
- Spanish-only.
- No data tracking, no SLP-facing tools.
- Single foundation, single-developer; cadence of updates is unclear.
- Pictogram library is small relative to [ARASAAC](arasaac.md) integrations.
- No cloud sync, no backup story.
- Not in active marketing; discovery only via lists like "best Spanish autism apps".

# Evidence base
- None published.

# Geographic / language coverage
- Latin American Spanish.
- Available via Google Play.
