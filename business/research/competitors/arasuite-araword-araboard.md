# What it is
Free open-source desktop suite from ARASAAC / Government of Aragón. Three legacy apps bundled: AraWord (pictogram word processor — type a sentence, the program writes pictograms above each word), AraBoard (communication board builder + player), TICO (interactive board authoring). Generates printable visual supports and on-screen AAC.

# Pricing
- Free, open source. Windows, Linux, web. Android version of AraWord exists.

# Features
- AraWord: simultaneous text + pictogram processor; instant ARASAAC pictogram lookup; export to print or PDF.
- AraBoard Constructor + Player: build communication boards from 1×2 up to 4×8 cells; play with TTS.
- TICO: build interactive boards with branching.
- Direct ARASAAC server integration: live pictogram lookup with associated voice.
- Custom image and audio import.

# Strengths
- Spanish-first, public-sector funded, no business risk.
- Foundation product for Spanish-speaking SLPs and special-ed teachers — cited as default in most Spanish-language autism resource lists.
- Portable artifacts: prints become physical boards a kid can use offline at home or school.
- Authoring tool model — more flexibility than fixed-app AAC.

# Weaknesses / gaps
- Desktop-first; mobile experience is limited and poorly maintained.
- Authoring model has a learning curve; not a turn-key parent product.
- UI is dated and inconsistent across the three apps.
- Documentation primarily in Spanish; videos abundant on YouTube but unstructured.
- No analytics, no user accounts, no cloud sync.
- Distribution via SourceForge / Aula Abierta — discoverability issues.
- Maintained by a small team within ARASAAC; release cadence is slow.

# Evidence base
- Decade-plus track record in Spanish public schools; no formal RCT.

# Geographic / language coverage
- Multi-language UI (Spanish, English, French, Portuguese, etc.); pictogram set is multilingual.
- Available worldwide; default audience is Spanish-speaking.
