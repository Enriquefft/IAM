# What it is
Cross-platform cloud AAC: the only mainstream AAC product that runs on iOS, Android, web, ChromeOS, Windows, Mac without a native rewrite penalty. Open source (AGPL-3.0). The user gets symbol-based communication; supporters (parents, SLPs) get a separate, free supporter login that can model on a second device, view usage data, and message the team.

# Pricing
- Cloud subscription — USD 9/month per communicator.
- Lifetime per-communicator — USD 295 one-time.
- Free 2-month trial.
- Supporters (parents, therapists, teachers) — free when linked to a communicator.

# Features
- Multi-platform: web, iOS, Android, Mac, Windows, ChromeOS.
- Open-source codebase on GitHub.
- OpenBoards file format for cross-app board portability — initiated this standard.
- Built-in usage data logging (which words at which time of day) with reports.
- Modeling mode for supporters on a second device.
- Secure team messaging (parents and SLPs around one communicator).
- Offline mode with cloud sync.
- Symbol library: SymbolStix, OpenSymbols, [ARASAAC](arasaac.md), others.
- Switch access, eye-gaze.

# Strengths
- The only mainstream AAC product with cross-platform parity at this price.
- Native data on use — closest competitor today to a "passive metrics" AAC story.
- Open source: data export is genuine, family is not locked in.
- Supporter accounts free — turns the parent and the SLP into first-class users without per-seat cost.
- OpenBoards portability is a real interoperability win.

# Weaknesses / gaps
- UI/UX is engineer-grade — visibly less polished than Proloquo2Go or TouchChat. Onboarding is rough for non-technical parents.
- Brand recognition outside the AAC enthusiast community is low. SLPs in private practice still default to Proloquo2Go.
- TTS quality varies by platform (uses native OS voices), inconsistent with hardware.
- Pre-built page sets are utilitarian; no marquee vocabulary like Crescendo or Unity.
- LATAM distribution is essentially "translate yourself"; no Spanish-first marketing or training.
- The "data and reports" product is present but shallow — counts of word usage, not behavioral or skills-acquisition data.

# Evidence base
- Open-source community; case studies on closingthegap.com but no RCT.

# Geographic / language coverage
- UI translated to ~30 languages including Spanish, Portuguese; quality is community-dependent.
- ARASAAC integration provides Spanish pictograms.
- Sold worldwide; weak distribution outside US.
