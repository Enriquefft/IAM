# What it is
ABA practice-management and data-collection platform. Notable for being the only ABA-tech vendor with research-grant funding (NIH SBIR), which obligates outcomes evidence. Publishes pricing publicly (rare in the category).

# Pricing
- Data Collection or Practice Management — USD 24/learner/month.
- All-in-one (both) — USD 48/learner/month.
- Bulk discounts.

# Features
- Data collection (DTT, behavior, task analysis, frequency, rate, duration, ABC, anecdotal).
- 370+ pre-built program templates (academic, language, math, life skills).
- Scheduling.
- Billing.
- Credential tracking.
- Mobile app (iOS) for offline data collection.

# Strengths
- Pricing transparency.
- NIH-funded → outcomes-evidence pressure built into product.
- Per-learner pricing rewards small clinics.
- 370+ templates is a meaningful library.
- Cleaner UX than CentralReach; competitive on data-collection ergonomics.

# Weaknesses / gaps
- Smaller install base than CentralReach.
- US only.
- ABA-only.
- English only.
- Per-learner billing surprises growing clinics.
- No direct kid or parent product.

# Evidence base
- Tooling, not clinical claim, but research-grant-funded so outcomes work in pipeline.

# Geographic / language coverage
- English, US.
