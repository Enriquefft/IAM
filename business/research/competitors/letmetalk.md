# What it is
Free AAC talker app from German developer AppNotize. Pictogram-grid model: line up pictograms to form a sentence and play. Donation-financed. Initially Android, later iOS. Pulls from [ARASAAC](arasaac.md) pictograms (~9,000 in-app).

# Pricing
- Free. iOS and Android. Donation-supported.

# Features
- Pictogram sentence-building grid.
- 9,000+ ARASAAC pictograms bundled.
- Add own photos via camera.
- Offline — no internet required.
- TTS in many languages (uses OS engine).

# Features supported languages
- 10+ UI/voice languages: English, Spanish, French, Italian, German, Chinese, Portuguese (PT + BR), Arabic, Russian.

# Strengths
- Free across both major platforms.
- ARASAAC pictograms — Spanish-language families have a working symbol set out of the box.
- Offline operation — usable in clinics, schools, and rural settings without WiFi.
- Genuinely cross-platform.
- Strong fit for low-resource settings.

# Weaknesses / gaps
- UI is dated; one-developer feel.
- Donation-funded: cadence of updates is slow.
- No data tracking, no clinician dashboard, no parent dashboard.
- No grammar / vocabulary scaffolding.
- Discoverability poor outside Germany / EU.
- TTS quality OS-dependent.

# Evidence base
- None published.

# Geographic / language coverage
- 10+ languages including Spanish (Castilian + LatAm via OS voices).
- Free worldwide via Google Play / App Store / Amazon.
