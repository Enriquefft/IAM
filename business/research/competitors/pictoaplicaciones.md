# What it is
Spanish foundation in Santiago de Compostela publishing a suite of free pictogram-based tools for autistic kids: Pictocuentos (animated stories with pictograms), Pictoagenda (daily activity scheduling), Pictosonidos (sound learning), Pictoaprende, Pictojuegos. Originally a 2012 CSR project of Grupo Promedia, transferred to Fundación Pictoaplicaciones in 2020.

# Pricing
- All apps free, no ads. Web + Android primarily.

# Features
- Pictocuentos: pictogram-supported animated tales (Hansel & Gretel, Goldilocks, Little Red Riding Hood, Ugly Duckling) with comprehension activities; multi-language.
- Pictoagenda: daily activity routines via pictograms; visual progression markers.
- Pictosonidos: pictogram-sound matching.
- Multiple smaller pictogram-based games.
- Available in Spanish, English, Portuguese, French.

# Strengths
- Spanish-first, free, foundation-backed.
- Multi-app suite under one brand reduces parent decision fatigue.
- Story-based content gives narrative scaffolding most other Spanish apps lack.
- Multi-language (4 languages with native quality).
- 14-year track record.

# Weaknesses / gaps
- Web-first; mobile experience varies; iOS support inconsistent across the suite.
- Each app is small in scope; cumulative content is meaningful but each install feels light.
- No data tracking, no parent dashboard.
- No clinician access mode.
- UI dated.
- Discovery requires landing on pictoaplicaciones.com; weak app store ASO.

# Evidence base
- None published; institutional adoption in Spanish special-ed.

# Geographic / language coverage
- Spanish + English + Portuguese + French.
- Worldwide via web + Google Play.
