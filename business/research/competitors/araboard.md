# What it is
AAC tool from University of Zaragoza GIGA-Affective Lab (Sandra Baldassarri and Eva Cerezo). Two paired apps: AraBoard Constructor (build communication boards) and AraBoard Player (use them). Direct integration with the [ARASAAC](arasaac.md) pictogram server. Boards from 1×2 up to 4×8 cells. Cross-platform across Windows and Android.

# Pricing
- Free. No ads.

# Features
- Constructor for board authoring; Player for end-user execution.
- Live ARASAAC pictogram lookup.
- Configurable grid sizes (1×2 to 4×8).
- Custom image and audio per cell.
- Windows and Android.

# Strengths
- Spanish university-built; long track record in Spanish public schools.
- Direct ARASAAC API integration.
- Free.
- Constructor + Player split is clean — therapist authors, kid uses.

# Weaknesses / gaps
- iOS missing.
- Old; UI from circa 2012.
- Authoring tool — assumes a supporter to build; not a parent-quick-start.
- Slow update cadence; primarily a research-output product.
- No analytics, no cloud sync, no multi-user accounts.
- Limited grid size compared to mainstream AAC.

# Evidence base
- Affective-Lab academic publications on AraBoard usability; no clinical RCT.

# Geographic / language coverage
- Spanish primary; multilingual labels via ARASAAC.
- Free worldwide.
