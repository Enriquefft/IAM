# Question
What sleep and daily-behavior patterns differentiate autistic children from non-autistic siblings using multi-sensor wearables, and how does device-based measurement compare to parent report?

# Method
- Open-science cohort study (Simons Sleep Project, SSP).
- 3,600+ days/nights of data per the cohort.
- Devices: Empatica EmbracePlus smartwatch, Dreem3 EEG headband, Withings sleep mat.
- Plus parent questionnaires and daily sleep diaries.
- Whole-exome sequencing available for all participants.

# Findings
- Digital devices show "higher accuracy and reliability compared to parent reports" of sleep.
- Autistic children show longer sleep-onset latencies than siblings.
- Longer sleep latency associated with behavioral difficulties in all participants regardless of diagnosis.

# Limitations
- US-only sample.
- Children 10-17 only (not toddlers).
- Idiopathic autism only (excluded syndromic).
- Cross-sectional baseline; not interventional.

# Quotes
> "Digital devices have higher accuracy and reliability compared to parent reports."

# Builder takeaways
- Largest open-data autism wearable cohort to date. Data available via SFARI Base.
- Confirms wearable feasibility in autistic adolescents — but EmbracePlus per-unit + cloud cost is incompatible with LATAM consumer pricing.
- Sleep latency is a tractable target: a software bedtime-routine product that improves a measurable signal would have rationale aligned with this evidence.
