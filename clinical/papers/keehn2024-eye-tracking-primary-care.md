# Question
Can a composite eye-tracking biomarker, integrated with primary care pediatrician judgment, improve diagnostic accuracy for autism in toddlers?

# Method
- Prospective diagnostic study with blinded eye-tracking index test and reference standard by clinical psychologist (ADOS-2, VABS-3, MSEL).
- Composite biomarker: 5 paradigms (nonsocial preference, attentional disengagement, pupillary light reflex, oculomotor metrics).
- Integrated model adds PCP diagnosis + diagnostic certainty.

# Findings
- Composite alone: sens 77.5% (95% CI 68.4-84.5), spec 77.3% (63.0-87.2), PPV 88.8%, NPV 59.6%.
- Integrated with PCP: sens 90.7%, spec 86.7%, PPV 95.7%, NPV 74.3%.
- Six eye-tracking indices differentiated children with autism.

# Limitations
- Single network; needs multisite replication.
- Reference standard psychologist time still required for confirmation.
- Hardware-dependent.

# Quotes
> Eye-tracking biomarkers "substantially improved diagnostic accuracy when integrated with primary care clinician judgment"

# Builder takeaways
- Strongest 2024 evidence that eye-tracking + clinician judgment beats either alone — supports a hybrid digital/clinical screening flow.
- Implication for LATAM: where pediatricians lack autism training (PE/EC), a screener can compensate; in markets with stronger primary care, the integration is the value.
- Hardware barrier: research-grade eye trackers ~$5K. Webcam alternatives untested at this fidelity. Worth pilot in PE.
