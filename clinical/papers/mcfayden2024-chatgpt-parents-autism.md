# Question
How accurate, clear, actionable, and reference-trustworthy are ChatGPT-4 responses to parent queries about autism?

# Method
- 13 parent-focused queries across three domains: basic info (7), myths/misconceptions (3), resources (3).
- Dual responses scored using "3Cs" framework (correctness, clarity, conciseness), understandability, actionability, language type, reference accuracy.
- Authors are autism researchers/clinicians (UNC).

# Findings
- Average 3C score: 3.59–3.69 / 4.0 — largely correct, concise, clear.
- Understandability: 78%.
- Hyperlink accuracy: only 42% functional — majority of cited URLs were inaccurate.
- Actionability: 0% for basic info/myths; 44% for resources — minimal practical guidance.
- Language predominantly medical, not neurodiversity-affirming.
- ChatGPT effectively debunked common autism myths.
- Knowledge cutoff: sources updated only through 2021.

# Limitations
- Only 13 questions; cross-sectional snapshot of ChatGPT-4 at one date.
- No comparator (vs physician, vs textbook).
- No measure of harm to readers or behavior change.
- Single rater pair; no large independent rater pool.

# Quotes
> "Largely correct, concise, and clear" but with "inaccurate references and hyperlinks." — Results

# Builder takeaways
- Off-the-shelf ChatGPT for parent autism Q&A delivers correct-sounding, medically-framed text with broken links — a pattern that erodes parent trust over months.
- Reference accuracy is the failure point a wellness product can solve: pin all cited resources to a curated, verified library and never let the model hallucinate links.
- Default GPT language is medical-deficit framing — neurodiversity-affirming output is achievable but requires explicit prompting/RAG-grounding.
- Actionability gap (0% on basic info) is the differentiator: turning "what is autism" responses into "here are 3 next steps in your country" is a defensible product wedge.
